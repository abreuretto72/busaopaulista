6. Tela do monitoramento. O aplicativo consulta periodicamente a base de dados do SPTrans para obter novas coordenadas do Carro selecionado. Toda vez que recebemos uma nova coordenada o mapa é atualizado. Caso o carro esteja próximo a parada selecionada o aplicativo vibra o aparelho e emite um som de buzina. Ás vezes, o monitoramento é interrompido pelo SPTrans(não sabemos a razão) fazendo com que o aplicativo avise da aproximação mesmo não estando próximo.
